# AgriVision Drone - AI Crop Disease Detection

## Problem
Farmers detect crop diseases too late, causing major crop loss.

## Solution  
Web app using Drone images + AI to instantly detect crop diseases.

## How to Run
1. Install: pip install -r requirements.txt
2. Run: python agri_drone.py
3. Open the Gradio link in browser

## Tech Stack
Python, Gradio, PIL, Pandas, YOLOv8 Simulation

## PEAS Framework
P - Performance Measure: Accuracy, Response time, Confidence score
E - Environment: Farm fields, Drone captured leaf images
A - Actuators: Display result, JSON output, Recommendations
S - Sensors: Camera/Drone Image Input
